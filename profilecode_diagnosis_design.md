# Profilecode独自診断 設計書

## 📋 概要

Profilecode独自診断は、MBTIや16PFとは異なる側面を測定し、実用的な自己分析ツールとして差別化します。
就職、転職、スキルアップ、自己研鑽、恋愛など、様々な場面で活用できる包括的な診断を提供します。

---

## 🎯 診断の目的と差別化

### 既存診断との違い

| 診断タイプ | 測定内容 | 用途 |
|-----------|---------|------|
| MBTI | 性格タイプ（4指標） | 性格理解 |
| 16PF | 性格因子（16因子） | 性格分析 |
| **Profilecode診断** | **実用的な能力・適性・スタイル** | **実生活での活用** |

### Profilecode診断の特徴
- ✅ **実用性重視**: 就職、転職、スキルアップ、恋愛などで直接活用できる
- ✅ **多面的分析**: 複数の側面から包括的に分析
- ✅ **アクション指向**: 診断結果から具体的な行動指針を提供
- ✅ **差別化**: 既存診断とは異なる側面を測定

## 🔍 診断次元の設計

### 提案1: 5次元包括診断（推奨）

#### 1. キャリア適性次元
**測定内容**: 仕事への適性、職種マッチング、働き方の好み

**サブ次元**:
- **職種適性**: 技術系、営業系、クリエイティブ系、管理系など
- **働き方スタイル**: チームワーク型、独立型、リーダー型、サポート型
- **価値観**: 安定性重視、成長重視、社会貢献重視、収入重視

**活用場面**:
- 就職・転職活動
- キャリアプランニング
- 職種選択

**結果例**:
```
あなたのキャリア適性タイプ: 「創造的イノベーター」
- 適性職種: デザイナー、エンジニア、起業家
- 働き方: 自由度の高い環境で創造性を発揮
- 価値観: 新しいことに挑戦し、イノベーションを起こす
```

#### 2. 学習スタイル次元
**測定内容**: 効率的な学習方法、知識の吸収パターン

**サブ次元**:
- **学習方法**: 視覚型、聴覚型、実践型、読書型
- **情報処理**: 論理的、直感的、体系的、体験的
- **学習環境**: 一人、グループ、オンライン、対面

**活用場面**:
- スキルアップ
- 資格取得
- 自己研鑽
- 教育方法の選択

**結果例**:
```
あなたの学習スタイル: 「実践的体験学習者」
- 最適な学習方法: 実際に手を動かして学ぶ
- 効果的な環境: 実践的なプロジェクト、インターンシップ
- 学習のコツ: 理論より実践、失敗から学ぶ
```

#### 3. コミュニケーションスタイル次元
**測定内容**: 人間関係の築き方、コミュニケーションの特徴

**サブ次元**:
- **コミュニケーション方法**: 直接的、間接的、感情的、論理的
- **関係性の築き方**: 深い関係、広い関係、距離感、親密さ
- **対立への対応**: 回避型、協調型、競争型、妥協型

**活用場面**:
- 恋愛・人間関係
- チームワーク
- コミュニケーション改善
- 人間関係の構築

**結果例**:
```
あなたのコミュニケーションスタイル: 「共感的リスナー」
- 特徴: 相手の気持ちを理解し、共感を示す
- 関係性: 深い1対1の関係を好む
- 対立対応: 協調を重視し、win-winを目指す
```

#### 4. ストレス対処次元
**測定内容**: ストレスへの対応方法、リカバリー能力

**サブ次元**:
- **ストレス反応**: 戦闘型、逃避型、適応型、無視型
- **リカバリー方法**: 休息、運動、趣味、人との交流
- **ストレス耐性**: 高い、中程度、低い

**活用場面**:
- ストレス管理
- メンタルヘルス
- ワークライフバランス
- 自己ケア

**結果例**:
```
あなたのストレス対処タイプ: 「適応的リカバリー型」
- ストレス反応: 状況に応じて柔軟に対応
- 効果的なリカバリー: 適度な休息と趣味のバランス
- 推奨行動: 定期的なセルフケア、境界線の設定
```

#### 5. 意思決定スタイル次元
**測定内容**: 判断の仕方、決断の特徴

**サブ次元**:
- **意思決定方法**: データ重視、直感重視、相談重視、時間重視
- **リスク許容度**: 高い、中程度、低い
- **決断速度**: 迅速、慎重、状況依存

**活用場面**:
- 意思決定の改善
- リーダーシップ
- 問題解決
- 選択の最適化

**結果例**:
```
あなたの意思決定スタイル: 「データドリブン型」
- 特徴: データと分析を重視して判断
- リスク許容度: 中程度（計算されたリスクは取る）
- 決断速度: 慎重だが、データが揃えば迅速
```

## 📊 診断タイプの分類

### 各次元のタイプ分類例

#### キャリア適性次元（8タイプ）
1. **創造的イノベーター**: 新しい価値を創造
2. **分析的専門家**: 深い専門性を追求
3. **社会的貢献者**: 人々の役に立ちたい
4. **組織的マネージャー**: チームをまとめる
5. **独立起業家**: 自分の道を切り開く
6. **安定実務者**: 安定した環境で働く
7. **成長挑戦者**: 常に成長を求める
8. **バランス型**: 複数の要素をバランス

#### 学習スタイル次元（6タイプ）
1. **視覚的学習者**: 図や映像で理解
2. **聴覚的学習者**: 音声や説明で理解
3. **実践的学習者**: 手を動かして理解
4. **読書的学習者**: テキストで理解
5. **体験的学習者**: 実際の経験から学ぶ
6. **統合的学習者**: 複数の方法を組み合わせ

#### コミュニケーションスタイル次元（6タイプ）
1. **直接的コミュニケーター**: 率直に伝える
2. **共感的リスナー**: 相手の気持ちを理解
3. **論理的説明者**: 論理的に説明
4. **社交的ネットワーカー**: 広い関係を築く
5. **深い関係構築者**: 深い1対1の関係
6. **適応的コミュニケーター**: 状況に応じて対応

#### ストレス対処次元（5タイプ）
1. **戦闘的対処者**: 積極的に問題解決
2. **適応的リカバリー型**: 柔軟に対応
3. **休息重視型**: 十分な休息を取る
4. **社会的サポート型**: 人との交流で回復
5. **無視回避型**: ストレスを無視・回避

#### 意思決定スタイル次元（5タイプ）
1. **データドリブン型**: データを重視
2. **直感重視型**: 直感を信じる
3. **相談重視型**: 他者の意見を参考
4. **迅速決断型**: 素早く判断
5. **慎重分析型**: 時間をかけて分析

## 🔄 質問マッピングの設計

### 質問の設計方針

1つの質問が複数の次元に影響するように設計：

```javascript
{
  id: 1,
  question: "新しいプロジェクトに取り組むとき、まず何をしますか？",
  options: [
    { text: 'チームメンバーと相談する', value: 1 },
    { text: 'データを集めて分析する', value: 2 },
    { text: '直感で進める', value: -1 },
    { text: '一人でじっくり考える', value: -2 }
  ],
  mappings: {
    profilecode: {
      career: { dimension: 'work_style', weight: 0.8 }, // チームワーク型
      learning: { dimension: 'learning_method', weight: 0.6 }, // グループ学習
      communication: { dimension: 'communication_style', weight: 1.0 }, // 相談重視
      decision: { dimension: 'decision_method', weight: 0.9 } // 相談重視
    },
    fourIndicator: {
      dimension: 'E/I',
      weight: 0.7 // 外向性
    },
    sixteenFactor: {
      dimension: 'warmth',
      weight: 0.5
    }
  }
}
```

## 💡 実用的なアドバイスの生成

### 総合アドバイスの例

各次元の結果を統合して、実用的なアドバイスを生成：

```
【あなたの総合プロフィール】

キャリア: 創造的イノベーター
学習: 実践的体験学習者
コミュニケーション: 共感的リスナー
ストレス対処: 適応的リカバリー型
意思決定: データドリブン型

【実用的なアドバイス】

1. キャリア面:
   - 創造性を活かせる職種（デザイナー、エンジニア）を検討
   - 自由度の高い環境を選ぶ
   - 新しいプロジェクトに積極的に参加

2. 学習面:
   - 実践的なプロジェクトで学ぶ
   - インターンシップやハッカソンに参加
   - 理論より実践を重視

3. 人間関係面:
   - 深い1対1の関係を大切に
   - 相手の気持ちを理解し、共感を示す
   - 対立時は協調を重視

4. ストレス管理:
   - 定期的なセルフケアを実施
   - 適度な休息と趣味のバランス
   - 境界線を設定する

5. 意思決定:
   - データを集めてから判断
   - 必要に応じて相談する
   - 計算されたリスクは取る
```

## 🎯 活用シーンの具体例

### 1. 就職・転職活動
- **キャリア適性**: 自分に合った職種を発見
- **学習スタイル**: 面接準備の効率的な方法
- **コミュニケーション**: 面接での伝え方
- **意思決定**: オファー選択の判断基準

### 2. スキルアップ・自己研鑽
- **学習スタイル**: 最適な学習方法を発見
- **ストレス対処**: 学習中のストレス管理
- **意思決定**: 学ぶべきスキルの選択

### 3. 恋愛・人間関係
- **コミュニケーション**: パートナーとの関係の築き方
- **ストレス対処**: 関係性のストレス管理
- **意思決定**: 関係性の判断

### 4. チームワーク・リーダーシップ
- **コミュニケーション**: チームでの伝え方
- **意思決定**: リーダーとしての判断
- **ストレス対処**: リーダーシップのストレス管理
